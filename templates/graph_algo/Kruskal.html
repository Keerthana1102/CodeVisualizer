Krushkals Algorithm

Krushkal algorithm is a greedy algorithm for solving the Minimum spanning tree problemSteps for finding MST using Kruskal’s algorithm are

  1. Sort all the edges in non-decreasing order of their weight.
  2. Pick the smallest edge. Check if it forms a cycle with the spanning tree formed so far. If cycle is not formed, include this edge. Else, discard it.
  3. While sptSet doesn’t include all vertices
    1. Pick a vertex u which is not there in sptSet and has minimum distance value.
    2. Repeat step#2 until there are (V-1) edges in the spanning tree.
    3. Union Find algorithm is used to detect cycles.
