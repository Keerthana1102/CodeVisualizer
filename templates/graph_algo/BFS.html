Breadth-First Search

BFS is a graph traversal algorithm. First the starting vertex is selected and it further moves onto its neighbours and if they are not visited yet; adds them into the tree. It continues until no vertex has non-visited neighbouring vertices.The steps are:

  1. Take a queue
  2. Push startvertex into queue
  3. Pop the queue and add push all neighbours of popped vertex that are not visited yet
  4. Continue until queue is empty

Complexity:
Best Average Worst Memory
O(1) O(V+E) O(V+E) O(V)
BFS(u), Q = {u}
while !Q.empty
for each neighbor v of u = Q.front, Q.pop
if v is unvisited, tree edge, Q.push(v)
else if v is visited, we ignore this edge
