Shortest Common Supersequence Algorithm

Readme

Implementation

The goal of the algorithm is to find the length of the shortest string that has both given strings as subsequences.The steps are:

  1. Find Longest Common Subsequence (lcs) of two given strings
  2. Insert non-lcs characters (in their original order in strings) to the lcs found above, and return the result.
  3. The idea is to follow the simple recursive solution, use a lookup table to avoid recomputations. Before computing result for an input, we check if the result is already computed or not. If already computed, we return that result.

m and n are length of both strings
