Implementation

The Knapsack problem is a problem in the combinatorial optimization. Here is discussed the 0/1 Knapsack problem. We are given a set of items each with a given weight and value, we have to determine the number of items to include (select either complete item or don't select it) such that the total weight is atmost the given weight and as large as possible.

Consider there are n items given. Let them be numbered from 1 to n, each with weight wi and value vi and the maximum weight capacity is W. Then,

maximize Σ vixi

subject to Σ wixi ≤ W and xi ∈ {0, 1}

where xi denotes if the item is to be included or not ie., if
xi = 0, item is not included, if xi = 1, then the item is included.
