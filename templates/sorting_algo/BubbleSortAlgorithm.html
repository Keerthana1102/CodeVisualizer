Bubble Sort

Readme

Implementation

Bubble sort, also referred to as sinking sort, is a simple sorting algorithm that repeatedly iterates through the list to be sorted, compares each pair of adjacent items and swaps them if they are in the wrong order (ascending or descending arrangement). The pass through the list is repeated until no swaps are needed, indicating that the list is sorted.
