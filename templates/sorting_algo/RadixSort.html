Radix Sort(Decimal Radix)

Description

Implementation

Radix Sort is best suitable sorting alorithm for smaller integers.The numbers are added to the buckets based on their digit place of consideration and then resorted until all digits places are covered.The steps are:

  1. Go through all elements and add them to their bucket lists based on digit's place of consideration
  2. now shift the digits place to left
  3. reorder the elements into the array in order from buckets
  4. Repeat above 3 steps reccursively until all digit places are coverec

Complexity:
Best/Average/Worst Memory Stability
O(log(max.num)) O(N) Yes
RadixSort(arr) {
const maxNum = Math.max(...arr) * 10;
let divisor = 10;
while (divisor < maxNum*10) {
      buckets = [...Array(10)].map(() => []);
     for (let num of arr)
          buckets[Math.floor((num % divisor) / (divisor / 10))].push(num);
     arr = [].concat.apply([], buckets);
     divisor *= 10;
}
   return arr;
}
