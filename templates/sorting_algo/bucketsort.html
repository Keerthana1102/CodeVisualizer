Bucket sort or bin sort, is a sorting algorithm that works by distributing the elements of an array into a number of buckets.

In this visualisation we are going to take input array as {11,9,21,8,17,19,13,1,24,12} and we are going to insert them in 3 buckets namely B1, B2, B3.

Each bucket is then sorted individually by different sorting algorithms.
The average time complexity for Bucket Sort is O(n + k).
