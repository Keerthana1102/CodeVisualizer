Selection Sort

Problem Description:
Sort given array of elements.
//Pseudo code for Selection Sort
for i in range(len(A)):
  min_idx = i
  for j in range(i+1, len(A)):
    if A[min_idx] > A[j]:
     min_idx = j

  A[i], A[min_idx] = A[min_idx], A[i]
