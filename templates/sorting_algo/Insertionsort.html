Insertion Sort

Readme

Implementation

Insertion sort is a simple sorting algorithm that works similar to the way you sort playing cards in your hands. The array is virtually split into a sorted and an unsorted part. Values from the unsorted part are picked and placed at the correct position in the sorted part.THe steps are:

  1. Iterate from arr[1] to arr[n] over the array.
  2. Compare the current element (key) to its predecessor.
  3. f the key element is smaller than its predecessor, compare it to the elements before. Move the greater elements one position up to make space for the swapped element.

Complexity:
Best Average Worst Memory Stability
O(n) O(n2) O(n2) O(n) Yes
insertionSort(int arr[], int n){
for (i = 1; i < n; i++){
     key = arr[i];
     j = i - 1;
     while (j >= 0 && arr[j] > key){
      arr[j + 1] = arr[j];
     j = j - 1;
  }
arr[j + 1] = key;
}
}
