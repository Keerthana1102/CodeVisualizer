Implementation


Largest Laexicographic array is found by making all permutation of the array and then pick the one which satisfies the condition of at most K swaps.
The time complexity of this approach is O(n!).
In greedy approach, first find the largest element present in the array which is greater than(if the 1st position element is not the greatest) the 1st position and which can be placed at the 1st position with at-most K swaps.
After finding that element, note its index.
Then, swap elements of the array and update K value.
Apply this procedure for other positions till k is non-zero or array becomes lexicographically largest.
for (let i = 0; i < n - 1 && k > 0; ++i){
      let indexPosition = i;
      for (let j = i + 1; j < n; ++j){
             if (k <= j - i) break;
             if (arr[j] > arr[indexPosition]){
                    indexPosition = j; }
      }
      for (let j = indexPosition; j > i; --j)
             SwapLets(arr, j, j - 1);
      k -= indexPosition - i;}
