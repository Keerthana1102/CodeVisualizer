Egyptian Fraction

Implementation


Every positive fraction can be represented as sum of unique unit fractions.
A fraction is unit fraction if numerator is 1 and denominator is a positive integer.
For a given number of the form ‘nr/dr’ where dr > nr, first find the greatest possible unit fraction, then recur for the remaining part.
Here Greedy paradigm is used that builds up a solution piece by piece, always choosing the next piece that offers the most obvious and immediate benefit
Such a representation is called Egyptian Fraction.
if (dr == 0 || nr == 0){return;}
if (dr % nr == 0) {

if (nr > dr){

let n = parseInt(dr / nr, 10) + 1;

printEgyptian(nr * n - dr, dr * n);
